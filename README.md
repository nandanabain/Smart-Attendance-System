# Smart Attendance System using Face Detection

## 📌 Description

This project is a Computer Vision-based attendance system that uses real-time face detection to automatically mark attendance using a webcam.

## 🚀 Features

* Real-time face detection using OpenCV
* Automatic attendance marking
* CSV file storage
* Simple and efficient implementation

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy

## ▶️ How to Run

1. Install dependencies:
   pip install opencv-python numpy

2. Run the program:
   python main.py

3. Press 'q' to exit

## 📂 Project Structure

Smart-Attendance-System/
│
├── dataset/

## 📊 Output

* Detects face via webcam
* Displays "Face Detected Successfully"
* Saves attendance in CSV file
